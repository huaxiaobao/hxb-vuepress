# markdown

markdown 语法贯穿我们整个项目，所以了解它的使用十分必要！
它的常用语法不多，可以边学边用

TIP

markdown 网址 (opens new window)

# front matter

这是专门为 markdown 做一些配置的一种语法格式,需要积累

基本规则：

  • front matter 必须是 markdown 文件中的第一部分
  • 并且必须采用在三点划线之间书写的有效的 YAML
  • 在写属性的时候，冒号后面必须空一格
    举例：
---
title: Blogging with VuePress
lang: zh-CN
---

常见的配置：

title:  文档标题
更新时间: 2023年8月7日
